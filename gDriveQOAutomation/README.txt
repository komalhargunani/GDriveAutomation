README
-------

Prerequisite - 
1. Install Java and maven 
2. Download the latest chrome driver and place it under ./drivers folder
3. Download the QO debug build and place it under ./extension folder, place the app folder inside it
4. Update the userName and password of the Google drive account in the config.properties file

How to run it - 
1. Go to the project directory on command prompt
2. Run - mvn test
